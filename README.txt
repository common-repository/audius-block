This is a simple Block Editor plugin that allows you to embed tracks from Audius.co using the track URL. 

== Description ==

Audius is a brand-new streaming platform built for all musicians, not just those signed to labels. 
Build a fanbase, share your works in progress, and then publish your completed tracks for all the world to hear.

This Wordpress plugin makes it incredibly easy to embed tracks from Audius.co using your favorite track URL. 
Easily choose beteween the compact and card embed layouts. Multiple Audius embeds can be added to your posts and 
pages; simply add a new block!

== Installation ==

1. Upload `audius-plugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Block Editor to add a new Audius block. 
4. Paste a link to one of your Audius tracks!

== Frequently Asked Questions ==

= Where do I find the track URL to paste? =

The Track URL can be located at the top of your web browser and should look similar to this: https://audius.co/brycedev/run-with-me-84800
